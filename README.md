# Whack-A-Mole
### A simple code exercise - [View demo](http://codepen.io/eddieebeling/full/wJVOMy/)

## Getting started

Clone repo, then run the following commands:

```shell
npm i && bower i
```

```shell
gulp serve
```

### Output Folder

All files publish to the `www` folder.

### Technolgoy

* AngularJS
* Coffeescript
* Gulp
* Sass
* Sass Media Queries
* Browser-Sync
